# Lovely Dance Of Karate 
![Project Screenshot]("D:\Unity Projects\Lovely Dance Of Karate\LDOK_icon.jpg")

A 2D time-trial based platformer!

## About the game

A 2D platformer where you have to find 4 cards across the map, before exiting towards the door, all while trying to be the fastest of them all.

## Features

* Fluid Movement: double jumps and wall jumps are inherent
* Inbuilt timer: a timer to see how fast you have beat the game
* Saved Best Time: locally saves your best time!
* A quick restart button: To replay to get a better time.

## Controls

|Move| Arrow Keys | WASD|

|Jump| Spacebar|
